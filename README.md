# EarTrainerEQ — плагин для тренировки технического слуха

Этот проект — VST3/AU плагин на **JUCE (C++)**. Он:

1. При каждом раунде выбирает случайную частоту из фиксированного списка 28 значений.
2. Случайно поднимает (+6 дБ) или опускает (−6 дБ) эту область с Q = 4.3 (peaking/bell фильтр).
3. Пользователь в интерфейсе плагина нажимает на кнопку с той частотой, которую, как ему кажется, изменили.
4. Плагин говорит «Верно» / «Неверно» и показывает счёт.
5. После ответа начинается новый раунд.

Ниже — максимально подробная инструкция, рассчитанная на то, что вы **никогда не собирали плагины**.

---

## 0. Что вообще происходит

VST-плагин — это скомпилированная под вашу ОС динамическая библиотека (.vst3 на Windows/Linux, .vst3/.component на macOS), которую DAW (Ableton, Cubase, Reaper и т.д.) загружает в себя. Чтобы её получить, нужно:

1. Установить компилятор C++ (Visual Studio / Xcode / GCC).
2. Скачать фреймворк **JUCE** — он даёт готовый код для аудио-обработки, GUI, и упаковки в форматы VST3/AU/AAX.
3. Взять исходный код, который я подготовил (папка `Source/`), и собрать его через **CMake**.
4. Скопировать получившийся `.vst3` в папку плагинов вашей DAW.

Это займёт примерно 30–60 минут в первый раз (в основном — установка инструментов).

## 1. Самый простой способ — сборка в облаке (ничего не ставить на компьютер)

Если не хочется устанавливать Visual Studio/Xcode и разбираться с git — используйте **GitHub Actions**. Это бесплатный сервис Github, который сам скачает JUCE и соберёт плагин на настоящих Windows и Mac серверах, а вам останется просто скачать готовый файл. Всё через сайт в браузере, без единой команды в терминале.

### Шаг 1 — Создать аккаунт и репозиторий
1. Зарегистрируйтесь на https://github.com (если аккаунта ещё нет) — обычная регистрация по email.
2. Нажмите зелёную кнопку **"New"** (или "+" в правом верхнем углу → "New repository").
3. Дайте название, например `EarTrainerEQ`. Оставьте репозиторий **Public** или **Private** (не важно). Нажмите **"Create repository"**.

### Шаг 2 — Загрузить файлы плагина
1. На странице только что созданного (пустого) репозитория найдите ссылку **"uploading an existing file"** (или кнопку "Add file" → "Upload files").
2. Разархивируйте присланный вам `EarTrainerEQ.zip` на компьютере — внутри будет папка `EarTrainerEQ` со всеми файлами (включая скрытую папку `.github`; показывать скрытые файлы может понадобиться включить в Finder: `Cmd+Shift+.`).
3. Перетащите **содержимое** папки `EarTrainerEQ` (файлы `CMakeLists.txt`, `README.md`, папку `Source`, папку `.github`) в окно загрузки на сайте — именно содержимое, а не саму папку `EarTrainerEQ` целиком.
4. Внизу страницы нажмите зелёную кнопку **"Commit changes"**.

### Шаг 3 — Дождаться автоматической сборки
1. Перейдите на вкладку **"Actions"** вверху репозитория.
2. Вы увидите запущенный процесс "Build VST3 Plugin" — он идёт автоматически после загрузки файлов. Сборка занимает 10-15 минут (компилируется JUCE + плагин сразу на Windows-сервере и Mac-сервере).
3. Когда рядом с процессом появится зелёная галочка (значит всё собралось без ошибок), откройте этот процесс кликом.
4. Внизу страницы будет раздел **"Artifacts"** с двумя файлами:
   - `EarTrainerEQ-windows-latest` — для друга (Windows)
   - `EarTrainerEQ-macos-latest` — для вас (macOS)
5. Скачайте нужный — внутри архива будет готовый `EarTrainerEQ.vst3`.

### Шаг 4 — Отправить другу
Просто отправьте другу файл `EarTrainerEQ.vst3` (мессенджером, почтой, облаком — как угодно). Ему нужно:
1. Скопировать этот файл в `C:\Program Files\Common Files\VST3\`
2. Пересканировать плагины в своей DAW.
Git, компиляторы и прочее ему не нужны вообще — только этот один файл.

Если появится красный крестик вместо галочки в Actions — значит сборка упала с ошибкой; кликните на неё, скопируйте текст ошибки и пришлите мне, разберём.

## 1а. Альтернатива — сборка локально (если не хотите использовать GitHub)

### Windows
1. Установите **Visual Studio 2022 Community** (бесплатно): https://visualstudio.microsoft.com/
   - При установке в списке компонентов обязательно отметьте **"Desktop development with C++"**.
2. Установите **CMake**: https://cmake.org/download/ (выберите Windows installer, при установке отметьте "Add CMake to PATH").
3. Установите **Git**: https://git-scm.com/downloads

### macOS
1. Установите **Xcode** из App Store (бесплатно).
2. Откройте Terminal и выполните:
   ```bash
   xcode-select --install
   ```
3. Установите **Homebrew** (если ещё нет): https://brew.sh
4. Через Homebrew поставьте CMake и Git:
   ```bash
   brew install cmake git
   ```

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install build-essential cmake git libasound2-dev libjack-jackd2-dev \
  ladspa-sdk libcurl4-openssl-dev libfreetype6-dev libx11-dev libxcomposite-dev \
  libxcursor-dev libxinerama-dev libxrandr-dev libxrender-dev libwebkit2gtk-4.0-dev \
  libglu1-mesa-dev mesa-common-dev
```

## 2. Скачивание JUCE

JUCE подключается как git submodule — то есть его исходники физически лежат рядом с вашим проектом.

Откройте терминал (на Windows — "Git Bash", которую поставил Git) и выполните:

```bash
cd /путь/куда/хотите/сохранить/проект
git clone https://github.com/juce-framework/JUCE.git
```

Это скачает JUCE (~500 МБ) в папку `JUCE` рядом с вашим проектом. Дождитесь окончания.

## 3. Структура проекта

Скопируйте папку `EarTrainerEQ`, которую я подготовил, **рядом** с папкой `JUCE`, так чтобы получилось:

```
МояПапка/
├── JUCE/                  <- то, что склонировали на шаге 2
└── EarTrainerEQ/          <- проект плагина (файлы ниже)
    ├── CMakeLists.txt
    └── Source/
        ├── PluginProcessor.h
        ├── PluginProcessor.cpp
        ├── PluginEditor.h
        └── PluginEditor.cpp
```

Все файлы уже готовы и приложены к этому сообщению — просто скачайте их и разложите как указано выше.

## 4. Сборка проекта

Откройте терминал внутри папки `EarTrainerEQ` и выполните:

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

Первая команда "настраивает" проект (генерирует файлы сборки под вашу систему — Visual Studio solution, Xcode project или Makefile). Вторая — компилирует.

Сборка первый раз может занять 10–20 минут, т.к. компилируется весь JUCE.

Если всё прошло успешно, .vst3 файл появится примерно тут:

- Windows: `build/EarTrainerEQ_artefacts/Release/VST3/EarTrainerEQ.vst3`
- macOS: `build/EarTrainerEQ_artefacts/Release/VST3/EarTrainerEQ.vst3`
- Linux: `build/EarTrainerEQ_artefacts/Release/VST3/EarTrainerEQ.vst3`

---

## 5. Установка в DAW

Скопируйте папку/файл `EarTrainerEQ.vst3` в стандартную папку VST3-плагинов вашей ОС:

- **Windows:** `C:\Program Files\Common Files\VST3\`
- **macOS:** `/Library/Audio/Plug-Ins/VST3/` (или `~/Library/Audio/Plug-Ins/VST3/` только для вашего пользователя)
- **Linux:** `~/.vst3/`

Затем в вашей DAW запустите пересканирование плагинов (обычно в настройках Plugin Manager) — и `EarTrainerEQ` появится в списке эффектов.

---

## 6. Как это работает изнутри (коротко)

- `PluginProcessor.cpp` — вся звуковая логика: держит список из 28 частот, в начале каждого раунда через генератор случайных чисел выбирает индекс частоты и направление (+6 дБ или −6 дБ), настраивает `juce::dsp::IIR` peaking-фильтр с этими параметрами и Q = 4.3, и в реальном времени фильтрует звук.
- `PluginEditor.cpp` — интерфейс: 28 кнопок с частотами, кнопка "Новый раунд", и текстовая метка с результатом ("Верно!" / "Неверно, правильный ответ — Х Гц"). При нажатии кнопки процессор сверяет её индекс с "правильным" индексом раунда.
- Оба файла подробно прокомментированы — можно менять тексты, цвета, размер окна и т.д.

## 7. Частые проблемы

- **"CMake not found"** — не добавили CMake в PATH при установке, переустановите с этой галочкой.
- **Долгая первая сборка** — нормально, компилируется весь JUCE framework.
- **Плагин не появляется в DAW** — проверьте, что .vst3 лежит именно в системной папке VST3 (см. шаг 5), и что вы нажали "пересканировать плагины".
- **Ошибки компиляции про JUCE не найден** — проверьте, что папки `JUCE` и `EarTrainerEQ` лежат строго рядом друг с другом (см. шаг 3).

Если что-то не соберётся — пришлите мне текст ошибки, разберём вместе.
